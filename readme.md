# 👕 Wardrobe Builder

Простой веб-конструктор образов с автоматическим удалением фона.

## 🛠 Запуск
1. Установи зависимости: `pip install -r requirements.txt`
2. Запусти сервер: `python app.py`
3. Открой: `http://localhost:5000`

## 📦 Структура
- `app.py` — Flask бэкенд + `rembg` (удаление фона)
- `templates/index.html` — фронтенд (drag&drop, слои, панель)
- `static/` — автоматически создаётся для хранения фото

## 🔄 Следующие шаги
- Сохранение готовых образов в БД
- Категоризация одежды (верх/низ/обувь)
- Экспорт лука в PNG
- Асинхронная обработка очередей
